# FCG.NotificationCenterLambda

Lambda .NET 8 responsável por consumir eventos assíncronos e simular o envio de notificações do sistema.

## Eventos suportados

- `UserRegistered`
- `PaymentProcessed`

## Variáveis de ambiente

- `APP_ENVIRONMENT`
- `NOTIFICATION_MODE`
- `NOTIFICATION_SENDER_NAME`
- `NOTIFICATION_SENDER_EMAIL`
